Professional personal portfolio website designed to showcase academic achievements, technical expertise, and featured projects. Built with a focus on minimalist aesthetics and robust performance.

## Technical Stack

| Domain | Technologies |
| :--- | :--- |
| **Frontend** | HTML5, Vanilla CSS, JavaScript (ES6+), Bootstrap 5 |
| **Backend** | C#, .NET Core, RESTful API |
| **Database** | PostgreSQL, SQL Server, MongoDB |
| **DevOps** | Docker, GitHub Actions |
| **Tools** | TortoiseGit, Postman, Azure, VS Code |

## Core Architecture

- **Internationalization (i18n)**: Implemented a robust client-side language switching system (VN/EN).
- **Dynamic Rendering**: Utilizes JavaScript data objects to manage and render skills, projects, and certifications dynamically.
- **Responsive Interface**: Optimized layout for seamless viewing across mobile, tablet, and desktop environments.
- **Minimalist Aesthetic**: Professional Slate and Blue design system focused on content clarity and recruiter UX.
